## Getting Started

### Prerequisites
Ensure you have the following installed:
* .NET 8.0 SDK
* Git

### Installation

1. Clone the repository
```sh
git clone https://gitlab.fdmci.hva.nl/project-se-dt/2324/team8/QuizTower-IDP.git
```

2. Navigate to the project directory
```sh
cd QuizTower-IDP
```

3. Restore NuGet packages
```sh
dotnet restore
```

4. Build the project
```sh
dotnet build
```
